# CodeAlpha - Customer Churn Analysis

## 📌 Project Overview
This project was completed as part of the **CodeAlpha Data Analytics Internship**. The objective of this project is to analyze customer churn data using Exploratory Data Analysis (EDA) and Data Visualization techniques to identify patterns and factors influencing customer retention.

## 🎯 Objectives
- Perform Exploratory Data Analysis (EDA)
- Clean and preprocess the dataset
- Analyze customer behavior and churn trends
- Create meaningful visualizations
- Generate business insights for customer retention

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights & Conclusion

## 📈 Visualizations
- Count Plot
- Bar Chart
- Histogram
- Box Plot
- Heatmap
- Correlation Matrix

## 📂 Dataset
The project uses the **Telco Customer Churn** dataset for analysis.

## 🚀 Key Insights
- Identified factors affecting customer churn.
- Analyzed customer demographics and service usage.
- Visualized important trends using charts and graphs.
